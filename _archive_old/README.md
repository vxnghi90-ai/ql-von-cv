**Hướng dẫn chi tiết các tính năng:** Xem file `TINH_NANG.md` (nên mở bằng trình duyệt hoặc VS Code / Typora để xem đẹp).

1. Double-click `Quản-Lý-Vốn-CV-Full.html`

Mọi dữ liệu lưu local trên máy bạn + nên xuất JSON backup thường xuyên.
